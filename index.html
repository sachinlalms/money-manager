To enable Google Sign-In for all users of your app:

  1. Go to console.cloud.google.com → New project
  2. APIs & Services → Enable Google Sheets API + Google Drive API
  3. Credentials → OAuth 2.0 Client ID → Web application type
  4. Add https://yourusername.github.io to Authorised JavaScript origins
  5. Copy Client ID and paste it into GSHEETS_CLIENT_ID in index.html

Until configured, the app works fully with local storage. Once set up, each user who signs in gets their own private Google Sheet — you cannot access their data.
